# Varun-firstc

A beginner C programming practice repository containing multiple small programs (`.cpp`) and their compiled outputs (`.exe`).

## Contents (Programs)

### 1) Hello World
**File:** `Varun First Code C.cpp`  
Prints a simple “Hello World” message.

### 2) Input a number and print it
**File:** `Varun C2.cpp`  
Takes an integer as input and prints it back.

### 3) Add two numbers (input)
**File:** `Varun c3.cpp`  
Reads two integers and prints their sum.

### 4) Add three numbers (input)
**File:** `Varun c4.cpp`  
Reads three integers and prints their sum.

### 5) Add two numbers (needs fixing)
**File:** `Varun c5.cpp`  
Intended to add two numbers, but currently has issues (e.g., `a` is not initialized and it prints `&b` instead of `b`). Treat this as a work-in-progress.

### 6) Add three numbers (input)
**File:** `Varun C6.cpp`  
Reads three integers and prints the sum in a formatted way.

### 7) Multiply two numbers
**File:** `Varun c7.cpp`  
Reads two integers and prints their product.

### 8) Multiply three numbers
**File:** `Varun C8.cpp`  
Reads three integers and prints their product.

### 9) Loops demo (for + while)
**File:** `Varun C10.cpp`  
Prints numbers 1 to 5 using a `for` loop and a `while` loop.

### 10) Square of a number (function)
**File:** `Vaarun c11.cpp`  
Uses a function `square(int)` to compute the square of an input number.

### 11) Basic arithmetic operations
**File:** `Varun c12.cpp`  
Demonstrates sum, difference, product, and quotient using two integers.

## How to compile and run

### Using GCC / MinGW (Windows) or GCC (Linux/Mac)
Compile any `.cpp` file like this:

```bash
g++ "Varun First Code C.cpp" -o program.exe
./program.exe
```

(Replace the filename with whichever program you want to run.)

## Note about `.exe` files
This repo includes compiled `.exe` files. Common practice is to keep only source code in Git and ignore generated binaries (optional).
